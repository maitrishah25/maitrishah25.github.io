John Deere

E-Commerce

E-commerce is considered a product family at John Deere, so it is not just one product but rather it has many. Below are some of the top UX initiatives.

  • Next Store Inventory: Enhanced fulfillment method that offers customer easier purchasing experience.
    Project link coming soon!
  • Dealer Selection: Machine learning mechanism that listens to signals provided by customers and match them with the best dealer.
    Project link coming soon!
  • Shop By Diagram: A way for experienced customers to find products using visual diagrams
    Project link coming soon!
  • Globalization: Localized experience based on the unique needs of global regions.
    Project link coming soon!
  • Deere Direct: Enhanced fulfillment method that enables John Deere to fulfill online purchases made while non-onboarded dealers are selected.
    Project link coming soon!
  • Tiered Portfolio: Original and remanufactured products made easily searchable.
    Project link coming soon!
  • Whole Goods: Beyond parts and merchandise, this marks the addition of equipment along with attachments and options to the product portfolio offered via John Deere e-commerce.
    Project link coming soon!
  • Externalized Checkout: Moving checkout off of the Hybris platform creates an opportunity to redesign of the checkout experience to increase the ease of use and reduce abandoned carts.
    Project link coming soon!
  • Promotions: Moving to a new promotions engine allows for reimagining the promotions experience to be more targeted, relevant and integrated into the customer shopping experience.
    Project link coming soon!
  • Universal Cart Integrations: This work enables the cart functionality to more applications within John Deere, including Maintenance Manager and Operations Center.
    Project link coming soon!
  • E-Commerce Process: I create a process and partner with the Group Product Manager to share documentation and implement the process across all e-commerce teams.
